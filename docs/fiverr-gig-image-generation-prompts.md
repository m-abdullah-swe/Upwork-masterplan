## Fiverr Image Guidelines

Use these practical specs:

- Canvas size: 1280 x 769 px.
- Orientation: horizontal landscape.
- Format: PNG or JPG.
- Keep text large and readable on mobile.
- Use no more than 6 to 10 words of large text on the main image.
- Use high contrast.
- Avoid clutter.
- Avoid fake Fiverr badges or misleading platform logos.
- Use original UI mockups or authorized portfolio screenshots.
- Use a consistent brand style across all gigs.

## Shared Visual Style

Use one consistent design system for all gig images.

### Recommended Style

- Clean modern SaaS/startup look.
- Dark navy or charcoal background.
- Bright accent color: electric blue, cyan, green, or violet.
- White headline text.
- Rounded app mockup cards.
- Subtle gradient glow.
- Realistic dashboard/mobile/web UI previews.
- Minimal icons.
- Professional but not corporate-heavy.

### Suggested Brand Palette

- Background: `#08111F` or `#0B1220`
- Primary accent: `#2F80ED`
- Secondary accent: `#00D1B2`
- Highlight: `#A855F7`
- Text: `#FFFFFF`
- Muted text: `#CBD5E1`

### Font Direction

Use clean bold sans-serif fonts:

- Inter
- Sora
- Manrope
- Poppins
- Montserrat

## Universal Negative Prompt

Use this negative prompt with every image-generation prompt:

```text
Avoid blurry text, misspelled words, distorted UI, fake Fiverr badges, excessive small text, cluttered layout, low-resolution screenshots, unrealistic hands, random code walls, stock photo clichés, watermarks, logos of companies not owned by me, App Store or Google Play badges unless provided, and misleading claims like guaranteed funding or guaranteed revenue.
```
